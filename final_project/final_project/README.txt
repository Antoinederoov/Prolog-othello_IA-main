Instructions for Launch:

* Version without HMI:

-> Alpha-Beta algorithm:
- Go to the 'console' folder
- Launch Prolog
- Load the 'skeleton.pl' => file? - [skeleton].
- Start othello =>? - init.

-> IDS algorithm:
- Go to the 'console' folder
- Launch Prolog
- Load 'skelete_ids.pl' => file? - [skelete_ids].
- Start othello =>? - init.

* Version with HMI (requires XPCE library):

*** NOTE: HMI is more suitable for Linux, so it is best to launch the HMI version from a Linux distribution

- Go to 'ihm'
- Launch Prolog
- Load the 'skeleton' file
